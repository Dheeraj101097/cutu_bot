# CUTU Bot Mobile App

Mobile-first UI for the CUTU Bot desktop companion. React + Vite + Tailwind + Framer Motion.

## Quick start

```bash
cd cutu-mobile-app
npm install
npm run dev
```

Open `http://localhost:5175` and use device toolbar (e.g. 375x667) for best preview.

## Features

- **Splash Screen** — Brand intro, auto-transitions
- **Landing** — Feature overview, Login / Sign up CTAs
- **Login & Sign Up** — Email, social, passwordless option
- **Bottom Tab Nav** — Bot, Images, Stocks, Music, More
- **Bot Chat** — Message input, image upload, save responses
- **Image Upload** — Gallery + camera, send to AI
- **Stocks** — Mock quotes, search, chart placeholder
- **Music Player** — Now playing, seek bar, playlist
- **More / Settings** — Profile, Notifications, Help, Feedback, About, Logout

## Design

- Light & cool palette (no dark mode)
- Glass-morphism cards, subtle blur
- Mobile breakpoints: 320px–480px
- See `STYLE_GUIDE.md` for colors, fonts, spacing.

## Tech stack

- React 19, Vite 7
- Tailwind CSS 3
- Framer Motion 12
- React Router 7
- Lucide React
